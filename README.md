#PySort 

Package Name:     marcinSortLite
main Module Name: marcinSort
Author:           Marcin Cherek
Language:         Python 2.7.11
Version:          0.01
Description
***********

This Package is for sorting lists. It provides three algorithms.
*HeapSort
*MergeSort
*QuickSort
*BubbleSort
*RadixSort

You can sort with a key or without.
